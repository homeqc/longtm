Program was tested in Ubuntu 12.08.
To test program do follow step below:
- Open Terminal
- Go to Program Directory.
- run make.

Some case test:
- Input character is not numeric which different 'Enter' 'Space' '-' will be consider input error ==> need do test again or exit test.
- Input number great than 100 will be consider input error ==> need do test again or exit test.
- Input each number less than 100 in each line, then press 'q' to show result. After that can press 'r' to continous test or press other button to exit.

Note: Case the first line must have one number is not implement.
